# Weather-angularJS
Capable of showing information about weather, timezone etc 


## What you need
1. Openweather API
2. Timezone API
3. Wikipedia API

### Installation
Change <your-api-key> to your API key

```javascript
// controller.js
var apiKey = "<your-openweather-API-key>";
var apiKey2 = "<your-timezone-API-key";
var apiKey3 = "<your-wikipedia-API-key>";
```
